# Components Directory

This directory contains all React components organized by their purpose:

## Structure

### `/layout`
Layout components that define the overall structure of pages (headers, footers, sidebars, etc.)

### `/widgets`
Reusable widget components (cards, panels, info displays, etc.)

### `/core`
Core UI components (buttons, inputs, modals, etc.)

### `/dynamic`
Dynamic components that change based on state or data
